# MINERVAE V2 Static Site

A minimal Next.js + Tailwind institutional website for MINERVAE.

## Pages

- `/` Homepage
- `/about` Institutional About page
- `/chambers` Editorial Chambers page
- `/essay-template` Essay reading template

## How to use

1. Upload this project to GitHub.
2. Import the GitHub repository into Vercel.
3. Deploy.
4. Connect `minervae.org` in Vercel.
5. Keep Substack as the publication engine.
